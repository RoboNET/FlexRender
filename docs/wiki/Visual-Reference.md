# Visual Reference - Interactive Property Examples

This is a visual guide with rendered examples for all FlexRender properties and element types. Each page provides side-by-side comparisons showing how different property values affect the final rendered output.

Perfect for visual learners who want to see exactly how each property works before implementing it in their templates.

## Table of Contents

### Canvas Settings

- [[Visual-Canvas]] - Canvas `fixed` sizing mode, `background` color, and `rotate` post-render rotation

### Flex Layout Properties

These pages demonstrate the core flexbox layout properties with visual examples:

- [[Visual-Justify]] - `justify` property alignment (6 values: start, center, end, space-between, space-around, space-evenly)
- [[Visual-Align]] - `align` property alignment (5 values: stretch, start, center, end, baseline)
- [[Visual-Direction]] - `direction` property main axis control (4 values: row, column, row-reverse, column-reverse)
- [[Visual-Wrap]] - `wrap` property wrapping behavior (3 values: nowrap, wrap, wrap-reverse)
- [[Visual-Position]] - `position` property (3 values: static, relative, absolute) with inset placement and practical patterns

### Element Types

These pages show rendered examples for each element type and their specific properties:

- [[Visual-Text]] - Text element properties including `align`, `wrap`, `maxLines`, and `lineHeight`
- [[Visual-Separator]] - Separator styles (dotted, dashed, solid) and orientation (horizontal, vertical)
- [[Visual-Image]] - Image fit modes including `contain`, `cover`, `fill`, and `none`
- [[Visual-QR]] - QR code `errorCorrection` levels (L, M, Q, H) and custom colors
- [[Visual-Barcode]] - Barcode formats, `showText` property, and custom colors

## How to Use This Guide

1. Browse the visual examples to understand how properties affect rendering
2. Find the property value that matches your design needs
3. Copy the YAML syntax from the examples into your templates
4. Refer to the comprehensive reference pages for detailed specifications
